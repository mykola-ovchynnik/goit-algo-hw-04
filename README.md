Порівняння продуктивності алгоритмів сортування

Порівняльний аналіз алгоритмів сортування, тестування їх на різних наборах даних. Емпіричний аналіз підтвердив теоретичні оцінки складності алгоритмів і продемонстрував, що поєднання сортування злиттям та сортування вставками робить Timsort набагато ефективнішим.

1. Сортування злиттям: Ефективне для великих наборів даних, але має вищі накладні витрати через рекурсивні виклики та етапи злиття.
2. Сортування вставками: Добре працює на малих наборах даних, але стає неефективним зі збільшенням розміру набору даних.
3. Timsort: Поєднує переваги сортування злиттям та сортування вставками, що робить його дуже ефективним для широкого діапазону розмірів наборів даних.
